TSR‑310 Voice Control

Drivers can control devices that support voice commands using the Opus Code 0 format, which allows these devices to receive voice commands from a proprietary TSR‑310 touch screen remote. The ITsrVoiceControllable interface provides this functionality.

NOTE: Version 19.x of the SDK framework must be used when implementing TSR‑310 voice control. For more information on versioning, refer to SDK Framework Versioning.

Overview

ITsrVoiceControllable is a capability interface that can be added as a capability to a driver.

The following methods are provided by ITsrVoiceControllable:

Method Description and Use
BeginVoiceSession() Begins a voice command session with the device. Must be called prior to calling SendVoiceData(System.Byte[]).

This method sets "ReadyToBeginVoiceSession" to true and triggers a "ReadyToBeginVoiceSessionChanged" event to applications.
EndVoiceSession() Closes an existing voice command session with the device.

This method sets "ReadyToBeginVoiceSession" to false and triggers a "ReadyToBeginVoiceSessionChanged" event to applications.
SendVoiceData(byte[] audio) Sends voice data to the device in the proprietary TSR‑310 voice format. The voice data bytes include the TSR header and the Opus packet.

This method should be called only between the BeginVoiceSession and EndVoiceSession calls.

Sample Driver Implementation

This section describes the implementation of the ITsrVoiceControllable capability interface on a sample driver that supports the Opus Code 0 format for receiving TSR‑310 voice commands.

The provided sample driver uses a generic cable box driver (that supports Opus Code 0 format) to implement the ITsrVoiceControllable capability. For more information about using sample drivers provided by the SDK, refer to Sample Projects.

Add the ITsrVoiceControllable capability interface to the sample driver as shown in the code sample below.

Copy 
public class TsrVoiceControl : ITsrVoiceControllable, IDisposable
{
    private readonly SampleCablebox_device;
    private readonly Logger _logger;
    private bool _readyToBeginVoiceSession = true;

    public TsrVoiceControl(Logger logger, SampleCablebox device)
    {
        _logger = logger;
        _device = device;
    }

    public void BeginVoiceSession()
    {
        // Send begin voice Session request here.
        if (_logger.LoggingEnabled)
        {
            _logger.Debug("Sending Begin Voice Session Request");
        }
        _device.Protocol.SendVoiceSessionRequest(“BeginVoiceSession”);

        _readyToBeginVoiceSession = false;
        if (ReadyToBeginVoiceSessionChanged != null)
        {
            ReadyToBeginVoiceSessionChanged(this, new ValueEventArgs<bool>(_readyToBeginVoiceSession));
        }
    }

    public void EndVoiceSession()
    {
        // Send end voice Session request here.
        _device.Protocol.SendEndVoiceSessionRequest(“EndVoiceSessionRequest”);
        if (_logger.LoggingEnabled)
        {
            _logger.Debug("Sending End Voice Session Request");
        }
        _readyToBeginVoiceSession = true;
        if (ReadyToBeginVoiceSessionChanged != null)
        {
            ReadyToBeginVoiceSessionChanged(this, new ValueEventArgs<bool>(_readyToBeginVoiceSession));
        }
    }

    public bool ReadyToBeginVoiceSession
    {
        get { return _readyToBeginVoiceSession; }
    }

    public event EventHandler<ValueEventArgs<bool>> ReadyToBeginVoiceSessionChanged;

    public void SendVoiceData(byte[] audio)
    {
        // Remove TSR header information from data.
        byte[] transformedVoiceData = TransformVoiceData(audio);
        if (transformedVoiceData == null)
        {
            return;
        }

        //Send Voice data here.
        var voiceData = Encoding.GetEncoding("ISO-8859-1").GetString(transformedVoiceData, 0, transformedVoiceData.Length);

        if (_logger.LoggingEnabled)
        {
            _logger.Debug(string.Format("Sending voice data - {0}", voiceData));
        }
        _device.Protocol.SendVoiceData(voiceData);
    }

    private static byte[] TransformVoiceData(byte[] data)
    {
        if (data.Length <= 0)
        {
            return null;
        }

        byte initialByte = data[0];

        // Read the first 5 bits to know the header length.
        int headerLength = initialByte & 0x1F;

        byte[] transformedVoiceData = data.Skip(headerLength).ToArray();

        if (transformedVoiceData.Length <= 0)
        {
            return null;
        }

        return transformedVoiceData;
    }
}

Once the ITsrVoiceControllable capability interface has been implemented, add it to the Main driver class and register it with the capabilities cache as shown in the code sample below.

Copy 
//Voice control capability.
var voiceControl = new TsrVoiceControl(Logger, this);
Capabilities.RegisterInterface(typeof(ITsrVoiceControllable), voiceControl);
