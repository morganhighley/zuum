What's New?

Updates to the Crestron Drivers SDK and documentation are described below from newest to oldest.

October 21, 2025

  • Crestron Drivers SDK v26 Released
    • Crestron Drivers SDK v26 has been released publicly and can be downloaded from the Crestron website via the Resources page. Refer to the release notes on the software download page for detailed information about this release.
    • The API Reference has also been updated to reflect the public v26 SDK.
  • Documentation Updates
    • Updated SDK Framework Comparison Matrices and SDK Framework Versioning topics.
    • Updated Load a Driver with updates to the Crestron Device Driver Configuration Tool.

May 13, 2025

  • Crestron Drivers SDK v25 Released
    • Crestron Drivers SDK v25 has been released publicly and can be downloaded from the Crestron website via the Resources page. Refer to the release notes on the software download page for detailed information about this release.
    • The API Reference has also been updated to reflect the public v25 SDK.
  • Documentation Updates
    • Added Lock Test Plans topic.
    • Updated Create the Driver Project (V1 SDK) and Create a Project (V2 SDK) with additional required references to NuGet packages.
    • Added new Crestron Home Programming topic for V1 SDK with Crestron Home Events and Crestron Home Sequences subtopics and removed mention of Crestron Home Extension devices.
    • Updated Crestron Home Programming topic for V2 SDK to remove mention of Crestron Home Extension devices.
    • Updated note in Load a Driver with information about forward slashes for 4-Series® control system file paths.

April 14, 2025

  • Crestron Drivers SDK v25 Released (EAP)
    • Crestron Drivers SDK v25 has been released to early access program (EAP).
    • The API Reference has also been updated to reflect the v25 SDK.
  • Documentation Updates
    • Added support for Lock drivers (V2 only):
      • Added Lock Drivers topics and subtopics.
      • Updated Supported Device Types, SDK Framework Comparison Matrices, and SDK Framework Versioning topics.
    • Updated AV Switcher Drivers with new Add Capabilities to Inputs and Outputs section.
    • Updated Add Extension Support with updates to Language Translations and UI Bindings sections.
    • Updated Configuration Item Definitions to point to the C# API Reference documentation.
    • Updated Configuration Flow with new Implement the Configuration Flow section.
    • Updated Crestron Home Programming with new Dynamic Feature Support section and to add missing sample code to AvailableValuesProperty section.
    • Updated Add and Define Configuration Items with new Update Configuration Items and Configure Child Devices sections.
    • Updated Add Commands and Properties with new Dynamic Features section.
    • Updated GeneralInformation to correct explanation text for DriverVersion.
    • Updated Permitted VideoInOut Values, Permitted AudioInOut Values, Permitted StandardCommands Values, Input Connector Details, Device Outputs Definition, AudioInOut, and VideoInOut to reflect new Stream input/output.
    • Added new Compatibility ID Guidelines topic.

January 31, 2025

  • Documentation Updates
    • Added Camera Test Plans and Microphone Test Plans topics.
    • Updated Camera API Reference and Microphone API Reference topics.
    • Changed "Lights" device type to "Light" to match naming conventions for other device types in various topics.
    • Updated SDK Framework Comparison Matrices topic with latest Crestron Database version.

January 21, 2025

  • Crestron Drivers SDK v24 Released
    • Crestron Drivers SDK v24 has been released publicly and can be downloaded from the Crestron website via the Resources page. Refer to the release notes on the software download page for detailed information about this release.
    • The API Reference has also been updated to reflect the v24 SDK.
  • Documentation Updates
    • Added support for Camera and Microphone drivers (V2 only):
      • Added Camera Drivers and Microphone Drivers topics and subtopics.
      • Updated Supported Device Types, SDK Framework Comparison Matrices, and SDK Framework Versioning topics.
    • Divided Entity Model API Reference into type-specific subtopics.
    • Added Entry Device Test Plans topic.
    • Updated Load a Driver with latest steps to obtain the Crestron Device Driver Configuration Tool.

November 12, 2024

  • Crestron Drivers SDK v23 Released
    • Crestron Drivers SDK v23 has been released publicly and can be downloaded from the Crestron website via the Resources page. Refer to the release notes on the software download page for detailed information about this release.
    • The API Reference has also been updated to reflect the v23 SDK.
  • Documentation Updates
    • Added support for Entry drivers (garage doors and gates) and Network Switch drivers (V1 only):
      • Added Entry Drivers topic
      • Added Network Switch Drivers topics and subtopics
      • Added Network Switch Test Plans topic
      • Updated Supported Device Types, SDK Framework Comparison Matrices, SDK Framework Versioning, and Entity Model API Reference topics

September 19, 2024

  • Crestron Drivers SDK v22 Released

    • The API Reference has also been updated to reflect the v22 SDK.
  • Documentation Updates
    • Updated SDK Framework Comparison Matrices to reflect latest SDK versions.
    • Updated Crestron Home Programming topic (formerly Crestron Home Sequences) for the V2 SDK to include commands, properties, and events. The V2 SDK Crestron Home Events topic has been depreciated.

June 11, 2024

  • Documentation Updates
    • Updated SDK Framework Comparison Matrices to reflect latest SDK versions.
    • Removed information about the depreciated SIMPL test program from SIMPL Testing sections.
    • Updated any affected Driver Self-Test Plans with a generic instruction for SIMPL testing. Updated all PDF versions that are located within Resources.

May 29, 2024

  • Crestron Drivers SDK v21.01 Released

  • Documentation Updates
    • Updated SDK Framework Comparison Matrices to reflect latest SDK versions.
    • Updated Load a Driver to include detailed instructions for loading V1 and V2 drivers to control systems and SIMPL programs.

March 19, 2024

  • Crestron Drivers SDK v21 Released
    • Crestron Drivers SDK v21 has been released publicly within Crestron Home OS (v4.00) and .AV Framework (v6.32).

      • NOTE: The v21 SDK is not currently supported in SIMPL. SIMPL support will be added in an upcoming release of the v21 SDK.

  • Documentation Updates
    • Large scale content rearchitecture to support V1 (RAD Framework) and V2 (Entity Model) SDK architectures. Refer to SDK Architecture Versions for more information.
      • Existing RAD Framework content (including the existing C# reference documentation) has been moved to Driver SDK V1 (RAD Framework).
      • New Entity Model content (including an integrated API reference topic) is located in Driver SDK V2 (Entity Model).
