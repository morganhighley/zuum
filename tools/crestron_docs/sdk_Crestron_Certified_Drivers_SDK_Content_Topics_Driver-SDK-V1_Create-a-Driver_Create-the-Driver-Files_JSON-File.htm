JSON Data File

The JSON data file contains a JSON string generated by the Data File Helper, which is used by the device.

In this topic, the TestDriver.json file will be used as an example.

To create a new JSON data file:

  1. Create and save an empty TestDriver.json file to a location on your workstation.
  2. Add the JSON file as a resource in the driver project.
  3. Mark the JSON file as an embedded resource.
