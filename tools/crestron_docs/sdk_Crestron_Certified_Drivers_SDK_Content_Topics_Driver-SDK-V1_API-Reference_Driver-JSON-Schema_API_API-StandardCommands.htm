StandardCommands

The StandardCommands property contains definitions for the majority of commands that will be used by the driver framework.

The StandardCommands property is of type Dictionary<StandardCommandsEnum, Commands> with the following key and value:

  • The key is the standard command.
  • The value is information about the standard command.

Only allowed standard commands may be used in the JSON. All other standard commands must be supported in the protocol class instead of overriding the appropriate method and building the command within the protocol.

Refer to the following example code:

Copy 
public override void PlayPause()
{
    CommandSet command=new CommandSet("PlayPause", "keypress/play", CommonCommandGroupType.Other, null, false, CommandPriority.Normal, StandardCommandsEnum.PlayPause);
    
    SendCommand(command);    
}

The value of type Commands is defined as follows:

Commands Properties

Property Use
StandardCommand

The standard command values of the StandardCommandsEnum that is used as the key.
Command The command string that will be sent to the device. Developers can insert extra characters and spaces here that can be replaced in PrepareStringThenSend, which is useful for volume commands.
Parameters No longer used.
