Entry Drivers

Any entity model driver that supports opening and closing an entryway may implement the entry capability. This capability would typically be implemented by devices such as garage doors, gates, and doors. The uxCategory indicates which type of entry device is being implemented.

Entry drivers may also implement additional capabilities. For example, a driver that controls a garage door with an integrated light could implement both the entry and light capabilities. If the driver will control multiple entry devices, then it should be a Platform Drivers device that exposes individual entry devices as managed devices.

To locate a self-test plan for entry device drivers, refer to Entry Device Test Plans.
