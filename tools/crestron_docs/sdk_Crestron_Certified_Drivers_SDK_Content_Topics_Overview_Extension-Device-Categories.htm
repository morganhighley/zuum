Overview / Supported Device Types

Supported Device Types

The Crestron Drivers SDK supports the following device types. When submitting a driver to Crestron for posting on the Crestron Drivers Portal, select one of the following categories that best matches the device type.

Native Device Types

The following device types have native support in certain Crestron applications, which provides a more integrated experience.

NOTE: Not all of the following device types have native support in all Crestron applications. Refer to the table below to determine whether the device type has native support within the provided applications.

Extension Device Types

Extensions are used to extend an application by providing their own custom user interface. Extensions are used to support a variety of device types, such as types that do not yet have native support, or types where Crestron does not plan to offer native support.
