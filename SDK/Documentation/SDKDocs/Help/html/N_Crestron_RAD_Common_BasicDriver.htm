ABaseDriverProtocol
ABasicDriver
The abstract class that represent all drivers. It provides a default implementation of the following component interfaces:

IBasicInformation, IConnection2, IBasicLogger, IBasicInformation2, IDisposable, ISupportedCommandsHelper, and IModelFileSupport.

Drivers should not override anything here and should instead override the protocol class unless specified otherwise.
