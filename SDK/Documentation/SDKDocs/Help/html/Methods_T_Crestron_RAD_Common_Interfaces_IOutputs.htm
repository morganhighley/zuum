IOutputs Methods

[This is preliminary documentation and is subject to change.]

The IOutputs type exposes the following members.

Methods
  NameDescription
GetUsableAudioOutputs
Gets array of the usable audio outputs on this Device.
GetUsableVideoOutputs
Gets array of the usable video outputs on this Device.
SetAudioOutputSource
Method to select an audio output source. AudioConnections is a superset of all possible outputs. For finding available outputs per device use GetUsableAudioInputs.
SetVideoOutputSource
Method to select an video output source. VideoConnections is a superset of all possible outputs. For finding available inputs per device use GetUsableVideoOutputs.
