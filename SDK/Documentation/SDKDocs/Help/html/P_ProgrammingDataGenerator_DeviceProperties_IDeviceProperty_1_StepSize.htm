IDevicePropertyTStepSize Property

[This is preliminary documentation and is subject to change.]

Amount to increment or decrement the value at a time; Applicable only when the Type refers to a numeric value.

If the type is Float or Double, this also defines the precision of the floating point value.

For example, if this value is 0.2, it would mean valid values could be 0, 0.2, 0.4, etc... Any invalid values passed to the [!:IDeviceWithProperties.SetValue<T>] method will be adjusted to the nearest valid value in the direction the value is being adjusted.
