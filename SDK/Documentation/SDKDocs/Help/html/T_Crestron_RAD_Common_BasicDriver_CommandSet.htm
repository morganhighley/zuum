CommandSet Class

[This is preliminary documentation and is subject to change.]

Represent a command that can be sent to the device. These can be instantiated by the driver and they can call SendCommand to send them to the device.

A name for the command that is used in logging.
CommandPrepared
Specifies if the command is already prepared. If overriding PrepareStringThenSend(CommandSet) then this must be checked prior to preparing the command. This overriden method can set this to true before calling the base method which will set it to true if it isn't already.
CommandPriority
The priority of the command. The higher the priority, the higher this command will be added in the sending queue.

Poll commands in most cases set this to Low.

Power commands in most cases set this to High.

IsPollingCommand
Parameters

StandardCommand
The standard command this command represents. This is used in the queue/sending logic.
SubCommandGroup
Primarly used for IAvrZone and IAudioVideoExtender, but can be used for cases where the CommandGroup doesn't provide enough details about the group of commands this command belongs to.

The queue/sending logic will use this instead of CommandGroup when this has a value other than Unknown.
