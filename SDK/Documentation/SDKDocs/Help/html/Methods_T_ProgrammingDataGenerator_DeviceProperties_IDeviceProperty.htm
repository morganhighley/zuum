The IDeviceProperty type exposes the following members.

Methods

GetStepSizeTValue
Retrieves the step size for this property. See StepSize for more details.

Also see GetValueTValue for more details about using these generic methods.

GetValueTValue
Retrieves the value of this property.

If the requested type does not match the type referred to by Type, an InvalidCastException will be thrown.

The only other allowed type is "object", with the caveat that it will cause value types to be boxed.
