Parameters

culture
Type: SystemString
Specifies the language code and country region code to be retrieved. The string must formatted as the "languagecode2" as defined by the ISO 639-1 standard, followed by a hyphen and then the "country/regioncode2" as defined by the ISO 3166.

For example, "en-US" for English in the United States.

If the specified culture is not supported, this will return null.
