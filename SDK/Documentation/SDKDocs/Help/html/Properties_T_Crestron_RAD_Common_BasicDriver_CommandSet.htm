CommandSet Properties

[This is preliminary documentation and is subject to change.]

The CommandSet type exposes the following members.

Properties
  NameDescription
AllowIsQueueableOverride
Avr Zone Field to allow for queuing of zone related commands
AllowIsSendableOverride
Avr Zone Field to allow for Sending of zone related commands
AllowRemoveCommandOverride
Avr Zone Field to allow for removing zone-related commands from the queue
CallBack
The Action that will be invoked after the command used to invoke Send on Transport.
Command
The command string that should be sent to the device.
CommandGroup
The command group of this command. This must be set to appropriate group for internal sending and receiving logic to function correctly.
CommandName
A name for the command that is used in logging.
CommandPrepared
Specifies if the command is already prepared. If overriding PrepareStringThenSend(CommandSet) then this must be checked prior to preparing the command. This overriden method can set this to true before calling the base method which will set it to true if it isn't already.
CommandPriority
The priority of the command. The higher the priority, the higher this command will be added in the sending queue.

Poll commands in most cases set this to Low.

Power commands in most cases set this to High.

IsPollingCommand
Parameters
Parameters that were set on the command that will be sent to the transport.
StandardCommand
The standard command this command represents. This is used in the queue/sending logic.
SubCommandGroup
Primarly used for IAvrZone and IAudioVideoExtender, but can be used for cases where the CommandGroup doesn't provide enough details about the group of commands this command belongs to.

The queue/sending logic will use this instead of CommandGroup when this has a value other than Unknown.
