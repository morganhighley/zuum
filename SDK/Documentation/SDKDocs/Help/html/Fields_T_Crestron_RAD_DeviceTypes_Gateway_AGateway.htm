The transport used for communication with the device which is responsible for transmitting and receiving messages.

DriverData

The common driver data between all device types. This will be set automatically by ConvertJsonFileToDriverData in each device-type library.

This should not be used or referenced - use the following instead:

Display - DisplayData

Cable Box - CableBoxData

Video Server - VideoServerData

Bluray Player - BlurayPlayerData

AV Receiver - AvrData
