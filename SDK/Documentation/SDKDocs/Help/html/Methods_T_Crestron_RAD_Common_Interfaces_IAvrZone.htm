IAvrZone Methods

[This is preliminary documentation and is subject to change.]

The IAvrZone type exposes the following members.

Methods
  NameDescription
GetUsableAudioInputs
Gets array of the usable audio inputs on this Device.
GetUsableVideoInputs
Gets array of the usable video inputs on this Device.
Mute
Method to send the Mute command to the Device.
MuteOff
Method to unmute the Device.
MuteOn
Method to mute the Device.
PowerOff
Method to turn the Device off.
PowerOn
Method to turn the Device on.
PowerToggle
Method to Toggle the Power State of the device.
SetAudioInputSource
Method to select an audio input source. AudioConnections is a superset of all possible inputs. For finding available inputs per device use GetUsableInput.
SetVideoInputSource
Method to select an video input source. VideoConnections is a superset of all possible inputs. For finding available inputs per device use GetUsableInput.
SetVolume
Method to turn the Device volume to a given percentage.
SetZoneSupportProperties
Method to get the supported features from the zone
VolumeDown
Method to turn the Device volume down one step.
VolumeUp
Method to turn the Device volume up one step.
