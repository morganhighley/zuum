DevicePropertyTAvailableValues Property

[This is preliminary documentation and is subject to change.]

Available values for the property; Null if not applicable.

The property’s value can only be set to one of these object's Value.

This may also contain labels for each value defined by MinValue through MaxValue. For example, if the value range is 0 to 3, this may define the labels for those values as "Off", "Slow", "Medium" and "Fast", respectively.
