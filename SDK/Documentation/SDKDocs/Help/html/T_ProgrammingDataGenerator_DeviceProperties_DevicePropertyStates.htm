Defines state information about a device property; Specified by the interface property State.

public enum DevicePropertyStates
Members
  Member nameValueDescription
Default0 Indicates a default, valid state.
Disabled1 Indicates if the property is currently disabled.

This may change based on the values of other properties. For example, changing the value of one property may disable another property.

ValueNotAvailable2 Indicates if the value of the property is not currently available.

In other words, the device may not have the value yet or the device is not capable of retrieving the value (for example, if a temperature sensor is not connected).

When this flag is present, the value returned by GetValueTValue or Value must be ignored.
