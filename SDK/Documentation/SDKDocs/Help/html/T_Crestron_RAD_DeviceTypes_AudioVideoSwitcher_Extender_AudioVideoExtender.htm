AudioVideoExtender Class

[This is preliminary documentation and is subject to change.]

Used internally for default IAudioVideoExtender logic. This class will decide if feedback is applicable to itself and relies on AAudioVideoSwitcherProtocol to do the processing so that a driver developer can override all methods within one class instead of creating a new implementation of IAudioVideoExtender to override certain fields. The same logic applies to commands being sent to the device.
