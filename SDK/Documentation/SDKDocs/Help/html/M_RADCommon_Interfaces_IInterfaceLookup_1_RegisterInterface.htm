IInterfaceLookupTBaseInterfaceRegisterInterface Method

[This is preliminary documentation and is subject to change.]

Recursively registers an implementation of an interface to be looked up later via TryGetInterface(Type). If the type passed is not an interface, the type itself is not added but the interfaces it implements are. Overwrites any previously registered interfaces of this type or base interfaces.
