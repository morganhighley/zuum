ABasicDriverUsernameKey Property

[This is preliminary documentation and is subject to change.]

When applications store a username value in CrestronDataStore with a specific key, this key can be supplied to the driver to use when authenticating with the device or when StoreUsername(string username) is called to store the username.

Driver developers - this must be implemented if the driver supports authentication with the device. base.UsernameKey must be called within the driver overridden method.
