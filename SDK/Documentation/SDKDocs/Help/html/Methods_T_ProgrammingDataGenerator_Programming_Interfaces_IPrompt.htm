The IPrompt type exposes the following members.

Methods
  NameDescription
GetAbsoluteMax
Determines a maximum value for this property even when a MaxValue attribute is not set.

Determines a minimum value for this property even when a MinValue attribute is not set.

(Inherited from IDeviceProperty.)
GetRuntimeValue
Translates the prompt stored value to what the target parameter needs. This will apply internal scaling values to map into the expected range of values for the method or property. Scaling values are set at construction.
GetStepSizeTValue
Retrieves the step size for this property. See StepSize for more details.

Also see GetValueTValue for more details about using these generic methods.

(Inherited from IDeviceProperty.)
GetValueTValue
Retrieves the value of this property.

If the requested type does not match the type referred to by Type, an InvalidCastException will be thrown.

The only other allowed type is "object", with the caveat that it will cause value types to be boxed.

(Inherited from IDeviceProperty.)
TryConvertAndSetValue
Attempts to convert the incoming value, provides the result, and retains it internally as the new default. This will factor in scaling calculations, will not restrict the incoming value to the Available Values listing and so anything can actually be assigned.
TryConvertValue
Attempts to convert the incoming value and provides the result. This will factor in scaling calculations.
ValidateRange
Helper method for the UI to be able to determine if a candidate value is out of range.
