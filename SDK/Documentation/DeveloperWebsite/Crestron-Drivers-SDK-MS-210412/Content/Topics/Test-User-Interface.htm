Test with the User Interface

Once the control system is configured with the test program and driver DLL files, you can begin to test drivers. The supplied interface for testing the driver is the Crestron XPanel touch panel emulator. It is a Windows® OS based application that emulates a Crestron touch screen.

In the Test Files folder of the SDK package, there is a folder labeled “User Interface”. In that folder, there are three files:

  • CrestronXPanel installer.air
  • CrestronXPanel installer.exe
  • rad_sdk_test_xpanel_<version>.c3p

If testing a driver for the first time, install XPanel by launching the CrestronXPanel installer.exe installer, and then open the rad_sdk_test_xpanel_<version>.c3p file.

Set Connection Settings

Connection settings for the control system must be set in XPanel prior to testing the driver:

  1. Open XPanel.
  2. Navigate to Options > Host Settings.

  3. Enter the IP address of the control system in the Hostname/IP Address field.

  4. Enter 03 in the IPID field if this value is not already present.
  5. Enter 41794 in the Port field if this value is not already present.
  6. Click Connect.
