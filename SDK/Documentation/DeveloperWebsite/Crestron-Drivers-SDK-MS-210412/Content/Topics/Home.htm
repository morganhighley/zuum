Crestron® Drivers: Developer Microsite

The Crestron® Drivers Software Development Kit (SDK) provides a platform for developing drivers to operate consistently and easily across Crestron solutions. The SDK allows manufacturers to integrate devices with various Crestron hardware, including 3‑Series® and 4-Series™ control systems, using popular communication protocols like IR, Serial, Ethernet and HDMI® (CEC).

With Crestron Drivers, recompiling a program to simply change a controlled device is no longer necessary. The platform is fully supported in SIMPL, SIMPL#Pro, Crestron Home™, Crestron Mercury®, and .AV Framework™ technologies.
