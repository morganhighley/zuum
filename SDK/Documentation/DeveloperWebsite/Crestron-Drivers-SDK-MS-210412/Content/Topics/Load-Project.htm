Load a Driver

The following procedures describe how to load a driver into the control system test program.

Visual StudioÂ® software compiles drivers as PKG files. The PKG file is essentially a ZIP file containing the following files:

  • <driver>.dll
  • <driver>.dat

The test program running on the control system uses the <driver>.dll file, which must be extracted manually from the PKG file.

Any driver to be tested must be loaded to the USER directory of the control system:

  1. Open the File Manager tool in Crestron Toolbox software.

  2. Click the pencil icon to display the Edit Address dialog box.

  3. Click the TCP radio button.
  4. Enter the control system IP address in the IP Address / Hostname field.
  5. Click OK. The File Manager tool updates to show the control system file system upon a successful connection.

  6. Expand the Internal Flash directory.

  7. Click the USER directory to display its contents. Any existing drivers display in the right-hand frame.
  8. Drag the <driver>.dll file from its location on your workstation to the USER directory. The <driver>.dll file displays in the right-hand frame if it is loaded successfully.

    9. NOTE: Multiple driver DLL files can be loaded to the USER directory at once.
