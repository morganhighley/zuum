Supported Interfaces

Supported driver interfaces can be selected from the Configuration menu of the user interface. At the top of each interface screen is a list of “Supports <feature>” labels that indicate the controls and feedback signals have been implemented in the device driver. These indicators drive which buttons and feedback signals actually display on the page.

Refer to the sections below for more information on each interface.

i-Basic Information

The i-Basic Information screen shows all available information for a device driver.

i-Connection

The i-Connection screen provides controls that can be used to force the control system to connect, disconnect or reconnect to the device under test. The connection status is also provided.

i-Power

The i-Power screen provides controls and feedback signals related to controlling the power of the device under test, including warm-up and cool-down (if supported).

i-Inputs

The i-Inputs screen provides controls and feedback signals related to controlling the input selections of the device under test.

i-Volume

The i-Volume screen provides controls and feedback signals related to controlling the volume of the device under test.

i-Lamp-Hours

The i-Lamp-Hours screen provides feedback signals related to the lamp hours of the device under test.

i-Video-Mute

The i-Video-Mute screen provides controls and feedback signals related to controlling the video mute of the device under test.

i-Navigation

The i-Navigation screen provides controls and feedback signals related to controlling the navigation of the device under test.

i-Keypad

The i-Keypad screen provides controls and feedback signals related to controlling the keypad of the device under test.

i-Media-Transport

The i-Media-Transport screen provides controls and feedback signals related to controlling the media transports of the device under test.

i-Channel

The i-Channel screen provides controls and feedback signals related to controlling the channel of the device under test.

i-DVR

The i-DVR screen provides controls and feedback signals related to controlling the digital video recorder (DVR) of the device under test.

i-Switching

The i-Switching screen provides controls and feedback signals related to controlling the audio and video switching of the device under test.

i-Camera-Control

The i-Camera-Control screen provides controls and feedback signals related to controlling the camera of the device under test.

i-Preset

The i-Preset screen provides controls and feedback signals related to controlling the presets of the device under test.

i-Logger

The i-Logger screen provides controls to enable extended logging for the device under test. Only the last debug message will be displayed on this page. However, using the Text Console in Crestron Toolbox, you can see all of the messages.

To view extended debugging messages in Crestron Toolbox:

  1. Open the Text Console tool in Crestron Toolbox software.
  2. Click the pencil icon to display the Edit Address dialog box.

  3. Click the TCP radio button.
  4. Enter the control system IP address in the IP Address / Hostname field.
  5. Click OK. The Text Console tool updates to show debugging messages if debugging is enabled.

i-Bluray-Player

The i-Bluray-Player screen provides controls and feedback signals related to controlling a Blu-ray Disc player device.

i-TV-Tuner

The i-TV-Tuner screen provides controls and feedback signals related to controlling a TV Tuner device.
